# vue3-admin

> 一个基于vue3的后台管理系统模版

[预览地址](https://wuwu00303.github.io/vue3-admin/)

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
